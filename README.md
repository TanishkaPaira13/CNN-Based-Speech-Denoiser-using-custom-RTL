# CNN-Based-Speech-Denoiser-using-custom-RTL
Lightweight CNN (2,481 parameters) for real‑time speech denoising, deployed on ZCU102 FPGA using custom Verilog RTL and AXI streaming. Achieves +7.65 dB SNR improvement and 13.39 dB noise floor reduction, with the entire model fitting in just 2.5 KB of on‑chip BRAM.
